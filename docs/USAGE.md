# CodeNova 编程语言教程说明文档

## CodeNova 编程语言简介

CodeNova 是一门内置在 熊猫编辑器 中的编程语言，具有简单易学、跨平台、高性能、中文编程、面向对象等特点。

CodeNova 具有以下特性：

- 简单易学：CodeNova 语言的语法可以和 Python 语言混合编写，学习成本低。
- 跨平台：CodeNova 语言可以在 Windows、Linux、Mac 等平台上运行。
- 高性能：CodeNova 语言的运行速度与 C 语言相当。
- 中文编程：CodeNova 语言支持中文编程，可以编写中文文档、注释、变量名等。
- 面向对象：CodeNova 语言支持面向对象编程，具有丰富的类、对象、方法等特性。


## 编写该 @帮助 代码

```python
@帮助
```

该代码将显示 CodeNova 编程语言的帮助文档。

## 2. 自动化库说明
**功能**：提供`自动化*()`函数，支持各类自动化操作（鼠标、键盘等）。

```python

```

### 支持的操作类型及使用方法

```python

账号输入框 = 界面*('输入框', 窗口对象, '请输入用户名', 100, 30, 180, 30)
密码输入框 = 界面*('输入框', 窗口对象, '请输入密码', 100, 80, 180, 30)

    确认 = 界面*('确认框', '提示', '确认登录?')
    如果 不是 确认: 返回
    如果 界面*('内容', 账号输入框) == "admin" 并且 界面*('内容', 密码输入框) == "123456":
        界面*('弹窗', '提示', '登录成功')
    否则:
        界面*('弹窗', '提示', '账号或密码错误')

界面*('按钮', 窗口对象, '登录', 100, 130, 100, 40, 登录)
界面*('运行', 窗口对象)  # 必须放在最后一行
```


## 6. 网络库说明
**功能**：提供`网络*()`函数，支持HTTP服务端（路由、启动）和客户端（GET/POST请求）操作。

### 导入方式
```python
@网络库
```

### 支持的操作类型及使用方法

| 操作类型   | 函数参数格式                                                                 | 代码示例                                                                 | 运行结果说明                                                                 |
|------------|----------------------------------------------------------------------------|----------------------------------------------------------------------|----------------------------------------------------------------------------|
| 注册路由   | 网络*('路由', 接口路径, 逻辑函数, 请求方法)（逻辑函数需为异步函数，接收请求数据并返回响应） | 网络*('路由', '/api/test', 处理函数, 'GET')                         | 注册GET接口`/api/test`，由处理函数处理请求                                  |
| 启动服务   | 网络*('启动', 服务器地址, 端口号)                                            | 网络*('启动', '127.0.0.1', 8000)                                    | 启动服务器，监听`127.0.0.1:8000`                                            |
| 发送POST   | 网络*('post', 请求地址, 请求数据)                                            | 网络*('post', 'http://127.0.0.1:8000/api/test', {'name': 'test'})   | 向目标地址发送POST请求，数据为`{'name': 'test'}`                            |
| 发送GET    | 网络*('get', 请求地址, 请求参数)                                            | 网络*('get', 'http://127.0.0.1:8000/api/test', {'id': 1})           | 向目标地址发送GET请求，参数为`{'id': 1}`                                    |
| 帮助       | 网络*('帮助')                                                              | 网络*('帮助')                                                        | 显示当前网络库帮助文档                                                       |

### 完整示例（服务端+客户端）
```python
@网络库
# 定义异步处理函数
异步 函数 get处理(请求数据):
    输出*(请求数据)  # 打印收到的请求数据
    返回 {'status': 'success', 'data': '已处理GET请求'}

异步 函数 post处理(请求数据):
    输出*(请求数据)
    返回 {'status': 'success', 'data': '已处理POST请求'}

# 注册路由
网络*('路由', '/api/get', get处理, 'GET')
网络*('路由', '/api/post', post处理, 'POST')

# 启动服务器
网络*('启动', '127.0.0.1', 8000)

# 发送请求
get_result = 网络*('get', 'http://127.0.0.1:8000/api/get', {'id': 100})
输出*(get_result)  # 打印GET响应

post_result = 网络*('post', 'http://127.0.0.1:8000/api/post', {'name': 'test'})
输出*(post_result)  # 打印POST响应
```

> 注：“杀死所有进程”可能导致系统崩溃，谨慎使用！


## 8. 随机库说明
**功能**：提供`随机*()`函数，生成各类随机数据（整数、浮点数、列表等）。

### 导入方式
```python
@随机库
```

### 支持的操作类型及使用方法

| 操作类型 | 函数参数格式                                                                 | 代码示例                                                                 | 运行结果说明                                                                 |

| 随机整数 | 随机*('整数', 最小值, 最大值)（包含首尾）                                    | 随机*('整数', 1, 10)                                                | 返回1-10之间的随机整数                                                     |
